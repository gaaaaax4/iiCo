# iiCo - 外部仕様書

## 1. アプリ概要

| 項目 | 内容 |
|---|---|
| アプリ名称 | iiCo |
| 対象プラットフォーム | iOS |
| 対象ユーザー | 幼児 |
| 概要 | ユーザーが登録した画像を単体でランダム表示して遊べるアプリ |
| 動作モード | フォアグラウンドで単体動作 |

---

## 2. 画面一覧

| 画面ID | 画面名 | 概要 |
|---|---|---|
| SCR-001 | Home画面 | アプリ起動直後に表示される導入画面 |
| SCR-002 | Play画面 | ランダムキャラクター表示画面 |

---

## 3. 画面仕様

### SCR-001 : Home画面

**表示タイミング**
- アプリ起動直後に表示される

**画面要素**

| 要素 | 種別 | 説明 |
|---|---|---|
| Start ボタン | ボタン | タップするとPlay画面へ遷移する |
| Start不可ログ | テキスト | 起動条件を満たさない場合、iiCoボタン下に理由を表示する |
| 画像ライブラリボタン | ボタン | 画像登録シートを開く |
| 登録画像サムネイル | UI | 登録済み画像の一部を横スクロールで表示する |

**画像登録シート仕様**

| 要素 | 種別 | 説明 |
|---|---|---|
| 画像を追加ボタン | ボタン | 写真ライブラリから任意の画像を複数選択して登録する |
| 登録枚数表示 | テキスト | 現在の登録枚数を 100 枚上限で表示する |
| 登録画像リスト | UI | 登録済み画像を縦スクロールで一覧表示する |
| 削除ボタン | ボタン | 登録済み画像を個別に削除する |
| 出現率セレクタ | UI | 画像ごとに出現率を 0〜100%（5%刻み固定）で設定する |
| エフェクト種別セレクタ | UI | 画像ごとに表示エフェクト種別（なし / バイブ）を設定する |
| 録音ボタン | ボタン | 画像ごとの音声を録音開始/停止する |
| 再生ボタン | ボタン | 画像に紐づく録音済み音声を再生/停止する |
| 音声削除ボタン | ボタン | 画像に紐づく録音済み音声を削除する |

**Startボタン有効条件**

| 条件 | iiCoボタン状態 | ログ表示 |
|---|---|---|
| 登録画像が1枚以上、かつ確率合計が100% | 有効 | なし |
| 登録画像が0枚 | 無効 | 「画像を1枚以上登録してください。」 |
| 確率合計が100%でない | 無効 | 「画像の確率合計を100%にしてください。」 |

**画面遷移**

```
Home画面 --[Startボタンタップ]--> Play画面
```

### SCR-002 : Play画面

**表示タイミング**
- Home画面のStartボタンタップ後に表示される

**画面要素**

| 要素 | 種別 | 説明 |
|---|---|---|
| 自分のキャラクター | UI | 起動時にランダムで割り当てられた登録画像がふわふわ揺れるアニメーションで表示される（表示サイズは従来比200%） |
| 背景色 | 画面全体 | 常時: 白 |
| 音声再生 | 音声 | 表示画像に録音済み音声がある場合、自動再生する |

**ランダム表示ルール**

| # | 条件 | 表示対象 |
|---|---|---|
| 1 | 画像ごとの出現率テーブルに当選 | 当選した登録画像を表示 |
| 2 | 登録画像が0枚 | 抽選不可（iiCoボタン無効） |
| 3 | 確率合計が100%でない | 抽選不可（iiCoボタン無効） |

**画像表示エフェクト**

| 項目 | 内容 |
|---|---|
| エフェクト種別 | なし / バイブレーション |
| 発火条件 | 登録画像が表示されたとき |
| 設定 | 画像ごとに種別をユーザーが任意に設定可能（バイブは強め・長め） |

**画像表示音声**

| 項目 | 内容 |
|---|---|
| 音声種別 | 画像ごとに録音した音声 |
| 発火条件 | 画像が表示されたとき |
| 設定 | 画像ごとに録音 / 再生 / 削除が可能 |
| 録音上限時間 | 1回あたり最大5秒（5秒経過で自動停止） |

**画面遷移**

```
Play画面 --[戻る操作]--> Home画面
```

## 4. ランダム表示の仕様

| 項目 | 内容 |
|---|---|
| 使用技術 | ローカル保存された画像データ |
| 動作端末 | 単体の iPhone 1台で完結 |
| 通信 | なし（完全オフライン） |
| ランダム選択ルール | 画像ごとの設定出現率テーブルで抽選 |
| 画像出現確率 | 画像ごとに 0〜100% をユーザー設定（5%刻み、合計100%以内） |
| 画像エフェクト設定 | 画像ごとに「なし / バイブレーション」を設定 |
| 画像音声設定 | 画像ごとに録音音声を設定 |
| 画像登録上限 | 最大100枚 |
| データ保存先 | Documents 配下（登録画像ファイル + メタデータJSON） |

### 必要な権限

| 権限 | 用途 |
|---|---|
| 写真ライブラリ | 任意画像の登録 |
| マイク | 画像ごとの音声録音 |

## 5. 確定した仕様

初期実装において以下の通り決定済み。

| # | 項目 | 決定内容 |
|---|---|---|
| 1 | 動作形態 | 完全オフライン（単体動作のみ） |
| 2 | ランダム対象 | 登録画像のみ |
| 3 | 背景色 | 白 |
| 4 | 登録画像の出現確率 | ユーザーが任意設定可能 |
| 5 | 画像表示エフェクト | 画像ごとに「なし / バイブ」を設定可能 |
| 6 | 最大登録枚数 | 100枚 |
| 7 | 画像表示音声 | 画像ごとに録音して表示時に自動再生 |
| 8 | 録音上限時間 | 5秒 |
