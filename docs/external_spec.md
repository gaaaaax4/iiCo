# iiCo - 外部仕様書

## 1. アプリ概要

| 項目 | 内容 |
|---|---|
| アプリ名称 | iiCo |
| 対象プラットフォーム | iOS |
| 対象ユーザー | 幼児 |
| 概要 | 2台のiPhoneが近づいたとき、双方の端末に音・画面変化によるフィードバックを与えるアプリ |
| 動作モード | フォアグラウンド・バックグラウンドの両方で動作 |

---

## 2. 画面一覧

| 画面ID | 画面名 | 概要 |
|---|---|---|
| SCR-001 | Home画面 | アプリ起動直後に表示される導入画面 |
| SCR-002 | Play画面 | 他端末との接触検知・フィードバック実行画面 |

---

## 3. 画面仕様

### SCR-001 : Home画面

**表示タイミング**
- アプリ起動直後に表示される

**画面要素**

| 要素 | 種別 | 説明 |
|---|---|---|
| Start ボタン | ボタン | タップするとPlay画面へ遷移する |

**画面遷移**

```
Home画面 --[Startボタンタップ]--> Play画面
```

---

### SCR-002 : Play画面

**表示タイミング**
- Home画面のStartボタンタップ後に表示される

**画面要素**

| 要素 | 種別 | 説明 |
|---|---|---|
| 自分の絵文字キャラクター | UI | 起動時にランダムで割り当てられた動物絵文字がふわふわ揺れるアニメーションで表示される |
| スキャン中アニメーション | UI | Bluetooth 準備完了後、拍動する円形アニメーションを表示する |
| Bluetooth 準備中表示 | テキスト | Bluetooth 初期化中はスピナーと「Bluetooth 準備中...」を表示する |
| 背景色 | 画面全体 | 通常時: 水色 / 接触時: オレンジ |
| 絵文字交換表示 | UI | 接触時に自分と相手の絵文字を並べて ↔ 矢印・星エフェクトとともに表示する |

**接触検知時の動作**

アプリの状態によって動作が異なる。

| # | アプリの状態 | アクション |
|---|---|---|
| 1 | 自端末・相手端末ともにフォアグラウンド | GATT で絵文字交換 → 双方で効果音 + 背景色変化（水色→オレンジ）+ 絵文字交換アニメーション |
| 2 | 自端末フォアグラウンド / 相手バックグラウンド | 自端末: 効果音 + 背景色変化 / 相手: ローカル通知（通知音）|
| 3 | 自端末バックグラウンド | 自端末: ローカル通知（通知音）/ フォアグラウンド復帰後に背景色変化 |

**接触後の復帰**

- 接触から **3秒後** に自動で通常状態（水色）に戻る

**画面遷移**

```
Play画面 --[戻る操作]--> Home画面
```

## 4. 近接検知の仕様

| 項目 | 内容 |
|---|---|
| 使用技術 | CoreBluetooth (BLE) |
| 対応機種 | Bluetooth 対応 iPhone（全機種） |
| 検知方式 | BLE の電波強度 (RSSI) で距離を推定 |
| 接触判定閾値 | RSSI ≥ -40 dBm（概ね 0〜5cm 以内） |
| 同時接触台数 | 1対1（2台） |
| 動作条件（自端末） | フォアグラウンド・バックグラウンドともに動作 |
| 動作条件（相手端末） | iiCo をインストール済みで、アプリがバックグラウンド以上で起動していること |
| 絵文字交換方式 | BLE GATT Characteristic（Read / Write）を使い双方の絵文字を交換する |
| 絵文字プール | 動物系絵文字 16 種類からランダム選択（起動時に決定）|
| 相手が完全終了の場合 | 動作不可（iOS の制約）|

### 必要な権限

| 権限 | 用途 |
|---|---|
| Bluetooth (常に許可) | 近接検知のためのスキャン・アドバタイズ |
| ローカルネットワーク | Bluetooth 検知の補助 |
| 通知 | バックグラウンド時の接触通知 |

## 5. 確定した仕様

初期実装において以下の通り決定済み。

| # | 項目 | 決定内容 |
|---|---|---|
| 1 | 接触判定の閾値 | RSSI ≥ -40 dBm（概ね 0〜5cm 以内）|
| 2 | 効果音 | iOSシステムサウンド（明るく短い音）|
| 3 | 背景色（通常時） | 水色 |
| 4 | 背景色（接触時） | オレンジ |
| 5 | 接触後の自動復帰時間 | 3秒 |
| 6 | 同時接触台数 | 2台（1対1）|

## 6. 今後の検討事項

| # | 項目 | 備考 |
|---|---|---|
| 1 | RSSI 閾値の調整 | 実機テストの結果を基に最適化する |
| 2 | 相手が完全終了時の対応 | APNs（プッシュ通知）を使えば対応可能だがサーバー実装が必要 |
| 3 | 3台以上のサポート | 現状は1対1のみ |
